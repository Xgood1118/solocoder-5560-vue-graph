## 1. 产品概述

通用图可视化编辑工具，面向知识图谱、流程图、网络拓扑、依赖关系等场景。用户可在浏览器中拖拽节点连边，实时查看图结构，并导出图数据或截图用于汇报。

- 目标用户：产品经理、架构师、运维工程师、数据分析师等需要可视化表达关系结构的人员
- 核心价值：零门槛在线图编辑，所见即所得，多格式导出，模板开箱即用

## 2. 核心功能

### 2.1 用户角色

| 角色 | 使用方式 | 核心权限 |
|------|---------|---------|
| 普通用户 | 直接打开网页 | 全部编辑与导出功能 |

### 2.2 功能模块

1. **编辑器主页**：SVG 画布、工具栏、属性面板、小地图、搜索栏
2. **节点创建面板**：双击空白处弹出，选择形状/图标/颜色
3. **属性面板**：右侧可编辑节点元数据（key-value）、富文本编辑

## 3. 核心流程

### 3.1 节点创建与编辑流程

用户双击画布空白处 → 弹出节点创建面板 → 选择形状/配色/图标 → 确认创建 → 节点出现在画布 → 单击选中 → 右侧属性面板可编辑文字与元数据 → 拖拽调整位置

### 3.2 连线与边编辑流程

从节点连接手柄拖出 → 拖到目标节点手柄释放 → 创建边 → 单击选中边 → 属性面板修改箭头方向/样式/弯曲度 → 双击边中段添加标签

### 3.3 分组与布局流程

框选多个节点 → Ctrl+G 编组 → 组可展开/折叠 → 组可嵌套 → 选择自动布局算法 → 节点自动排列

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深靛蓝(#1e1e2e)为画布背景，亮青(#00d4aa)为交互强调色，暖橙(#ff6b35)为警告/删除色
- **辅助色**：冷灰梯度用于面板与工具栏，半透明毛玻璃效果用于弹出面板
- **按钮风格**：圆角 8px，hover 时微发光效果，active 时内阴影
- **字体**：JetBrains Mono 用于标签/代码类文本，Noto Sans SC 用于中文界面
- **布局**：左侧窄工具栏 + 中间大画布 + 右侧属性面板，顶部搜索/操作栏
- **图标风格**：线性 Lucide 图标，2px 描边

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|---------|
| 编辑器主页 | SVG 画布 | 深色背景，点状网格，节点带阴影与发光选中态 |
| 编辑器主页 | 工具栏 | 左侧垂直图标栏，hover 显示 tooltip |
| 编辑器主页 | 小地图 | 右下角 200x150px 半透明缩略图，蓝色视口框 |
| 编辑器主页 | 属性面板 | 右侧 280px 宽面板，毛玻璃背景，可折叠区段 |
| 编辑器主页 | 搜索栏 | 顶部居中搜索框，实时过滤高亮 |
| 编辑器主页 | 右键菜单 | 深色毛玻璃下拉菜单，图标+文字 |
| 节点创建面板 | 形状网格 | 6 种形状卡片网格，hover 放大预览 |
| 节点创建面板 | 图标搜索 | 搜索框+图标网格结果 |
| 节点创建面板 | 颜色选择 | 预设色板 + 自定义取色器 |

### 4.3 响应式

- 桌面优先设计，最小支持 1280px 宽度
- 属性面板在窄屏时可折叠为抽屉
- 小地图可收起
